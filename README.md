tufuapi
=======

痛風関係の情報を表示するwebaアプリ


### clone
```
git clone https://github.com/mshige1979/tufuapi.git
```

### api作成
```
cd tufuapi
mojo generate app Tufu::Web
```

### 環境
- Linux(CentOS)
- Perl
- MySQL

- PhpStorm
- git
- Sequel Pro
